# 2XL Delauwe app 

## Launching the projet 

You should have the version of ruby indicated in the gemfile installed. Ruby version can be installed with: https://github.com/rbenv/rbenv. 

Launch server
bin/rails s

Install packages
bundle install

Install the right version of ruby : 
rbenv global 3.1.2
